# 文明6 Lua 辅助工具

查询 Civilization VI 的 Lua API 以及 Events 的辅助工具，免去模组作者频繁翻阅文档的麻烦。

## 下载

前往 [Releases](https://github.com/Aws63140/Civ6-Lua-Helper/releases) 下载最新版 `Civ6LuaHelper.exe`，双击运行即可。

## 使用说明

| 功能 | 说明 |
|------|------|
| 搜索 | 支持模糊搜索函数名和备注，多个关键词用空格分隔 |
| 筛选 | 按根对象、环境（UI/GamePlay）层级筛选 |
| 详情 | 显示函数签名、参数、返回值、备注、示例代码（Lua 语法高亮） |

## 更新

工具启动后点击 **帮助 → 检查更新**，自动从 GitHub Releases 检测新版本并下载。
